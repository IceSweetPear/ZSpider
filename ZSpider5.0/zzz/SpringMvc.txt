安装配置
	1，下载安装
		下载 java jdk1.8  Tomcat8.0 Eclipse EE  
		需要新的环境变量  JAVA_HOME 为 java 安装目录  CATALINA_HOME 为 tomcat安装目录  环境变量path中添加 JAVA_HOME  下的bin CATALINA_HOME 下的 bin  lib
 
	2,   搭建环境
		新建一个 动态web项目  （如果没有新建动态项目选项 help->install new software  安装 Web, XML, Java EE and OSGi Enterprise Development 重启就可以了）
		新建项目 选择 tomcat服务器如果找不到服务器 就点击 windows->Preferences->Server->Runtime Environments ->add 添加 Tomcat服务器
		（此处可能 不支持 Tomcat 8服务器需要   Help->eclipse marketplace ->find Eclipse JST Server Adapters->install 安装成功就好了）
		（如果你安装的 8.5的服务器 不支持 则需要  去 Tomcat安装目录 lib 下面 找到 catalina.jar 使用压缩工具打开 编辑版本）

	3，导入所需的Jar包
		在WEB-INF下的lib 导入 sping mvc 所需的jar包 （不要忘了 commons-logging.jar）
	
	4,   配置文件
		web.xml	在 WEB-INF 新建 web.xml  此文件为重要的配置文件 例如 拦截器的位置 
		springmvc.xml  如果web.xml 中配置的位置为 classpath:springmvc.xml  那么 在src 下新建新建 springmvc.xml 用来配置资源拦截器
	
	5，编写Controller
		在 src下新建包 新建java文件 在类上加注解 @Controller  在方法上  在方法上加 @RequestMapping("/helloworld") 用来映射路由 
		此方法返回的字符串 会在 拦截器定义的目录下 寻找同名的 文件

	6，编写Jsp
		在拦截器定义的目录下编写jsp文件
