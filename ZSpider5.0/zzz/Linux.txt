常用方法

进程管理
	ps		显示进程	
		-A 		所有的进程均显示出来
		-a 		不与terminal有关的所有进程
		-u 		有效用户的相关进程
		-x 		一般与a参数一起使用，可列出较完整的信息
		-l		较长，较详细地将PID的信息列出
	kill -sign *		终止进程
		-1		启动被终止的进程
		-2		相当于输入ctrl+c，中断一个程序的进行
		-9		强制中断一个进程的进行
		-15		以正常的结束进程方式来终止进程
		-17		相当于输入ctrl+z，暂停一个进程的进行

文件管理
	cd *		进入一个目录
	ls		查看目录结构
	ll		查看目录结构
	find *		查找一个文件
	| grep * 		并且条件
	pwd 		显示当前路径
		-P 		显示当前路径
	tar		压缩/解压 文件
		-j 		通过bzip2的支持进行压缩/解压缩
		-z 		通过gzip的支持进行压缩/解压缩
		-v 		在压缩/解压缩过程中，将正在处理的文件名显示出来
		-f  *		*为要处理的文件
	 gzip *		压缩文件
		-d		解压缩
文件操作
	touch *		新建一个文件
	mkdir		新建一个文件夹
	vim *		编辑一个文件
	rm  *		删除一个文件
		-r		递归删除一个文件
		-f 		force强制的意思，如果目标文件已经存在，不会询问而直接覆盖
	cp * *		复制文件
		-r		递归移动 可移动目录
		-f 		force强制的意思，如果目标文件已经存在，不会询问而直接覆盖
	mv * *		移动文件/重命名
		-r		递归移动 可移动目录	
		-f 		force强制的意思，如果目标文件已经存在，不会询问而直接覆盖
